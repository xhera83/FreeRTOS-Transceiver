Enumeration Type Documentation

◆ eMultiSenderQueue

enum eMultiSenderQueue

Enumeration is used to select a multi-sender-queue for different class methods.

Normally one would need to select a communication line by supplying a FRTTransceiver_TaskHandle. A multi-sender-queue does not have a unique FRTTransceiver_TaskHandle, so that another way of selecting a communication line had to be introduced.

Note
There should be a maximum of FRTTRANSCEIVER_MAXELEMENTSIZEONQUEUE enumeration keys.
Increasing FRTTRANSCEIVER_MAXELEMENTSIZEONQUEUE also requires one to adjust the amount of eMultiSenderQueue keys to match FRTTRANSCEIVER_MAXELEMENTSIZEONQUEUE
Enumerator
